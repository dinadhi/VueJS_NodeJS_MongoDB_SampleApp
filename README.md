# Instructions

``` bash
# Step 1 - Install mongodb, npm/yarn, nodejs in Operating system. 
# Step 2 - cd mevn 
# Step 3 - cd server
# Step 4-a - npm install / yarn (it will install all the server dependencies) 
# Step 4-b - npm start 
# Step 5 - It will show Connection established (if you have correctly install mongodb) 
# Step 6 - cd client Step 6-a - npm install / yarn (it will install all the client dependencies) 
# Step 7 - npm run dev Step 8 - and perform CRUD operation.

Posts are persisting in MongoDB, It will list posts and Sort it reverse order. (like the new Post at the top) It will update the Post. It will delete the Post. (if you want to see the Mongodb database Visually install MongoDB Compass)
```

# Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
